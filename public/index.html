글을 작성하면
실제 블로그 글 페이지가 생성되는 구조

이 MVP는 관리자 화면에서 글을 저장하면 D1에 데이터가 들어가고, 공개 페이지는 /post/slug에서 SSR HTML로 생성됩니다. 홈과 글목록은 정적이라 가볍고, 개별 글은 캐시까지 적용되어 빠르게 열립니다.

정적 + SSR 혼합 테스트

홈과 목록은 정적으로 빠르게 열리고, 개별 글만 SSR로 렌더링해 SEO와 속도를 함께 잡습니다.

템플릿형 작성

제목, 요약, 카테고리, 태그, 본문 마크다운만 넣으면 실제 글 페이지 HTML이 생성됩니다.

캐시 헤더 확인 가능

상세 페이지 응답 헤더의 x-blog-cache로 HIT / MISS를 바로 확인할 수 있습니다.
